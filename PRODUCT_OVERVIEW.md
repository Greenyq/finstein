# Finstein — AI-Powered Family Finance Advisor
## Product Overview for Investors

---

## Executive Summary

**Finstein** is an AI-powered personal finance management bot built natively inside Telegram — the messaging platform with over 900 million monthly active users. We help Canadian families track income and expenses, manage shared budgets, and receive personalized financial advice through simple conversational messages — no separate app required.

Our core insight: **the best financial tool is the one people actually use**. By living inside a chat people already open dozens of times a day, Finstein eliminates the friction that kills every traditional budgeting app.

---

## The Problem

**Personal finance is broken for the average family:**

- 63% of Canadians live paycheck to paycheck (FP Canada 2023)
- The average Canadian household spends 15+ hours per year manually logging finances
- Traditional budgeting apps (Mint, YNAB) have ~80% abandonment rates within 30 days
- Existing solutions require learning new interfaces, manual data entry, and continuous re-engagement
- Bilingual Russian-Canadian immigrant households (700,000+ people) are severely underserved

**The root cause:** Every existing solution forces users to change their behavior. We flip this — Finstein comes to where users already are.

## Our Solution

Finstein is a **bilingual (Russian + English) AI financial advisor** that lives entirely inside Telegram. Users manage their complete financial life through natural conversation:

> "Потратил 45 на кофе" → ✅ logged as $45 Coffee / Restaurant
> "How much did I spend on groceries this month?" → 📊 instant AI answer
> "Send me my monthly report" → 📈 full Claude AI analysis

No apps to download. No forms to fill. Just chat.

## Core Features

### 1. Natural Language Transaction Logging
Users record transactions the way they would text a friend — in Russian, English, or a mix of both. Our AI parser (Claude Haiku) interprets intent, extracts amounts, assigns categories, and confirms the entry in under a second.

**Supported input formats:**
- Text messages: "spent 120 on groceries" / "зарплата 3500"
- Voice messages: transcribed via OpenAI Whisper, then parsed
- Photo receipts: snap a receipt and log it instantly
- Excel / CSV files: bulk-import entire transaction histories

### 2. Intelligent Expense Categorization
Transactions are automatically sorted into a structured hierarchy:

| Group | Categories |
|-------|------------|
| **Income** | Paycheck, Child Benefits, EI, Cashback, Freelance, Gifts |
| **Fixed Needs** | Mortgage/Rent, Car Payment, Insurance, Subscriptions |
| **Variable Needs** | Groceries, Utilities, Gas, Pharmacy, Transportation |
| **Wants** | Restaurants, Entertainment, Shopping, Hobbies, Travel |
| **Savings** | Emergency Fund, RRSP, TFSA, Investments |

### 3. AI-Powered Financial Reports
Monthly reports generated by Claude Sonnet include:
- **Financial Health Score** (0–100) with interpretation
- **Spending Trend Analysis** — month-over-month comparison
- **Risk Identification** — cash flow warnings, overspending alerts
- **Savings Opportunities** — specific, actionable recommendations
- **Canadian-Context Tips** — TFSA/RRSP guidance, benefit optimization

### 4. Family Budget Collaboration
Finstein supports household budget sharing:
- One family account, multiple members (spouse, adult children)
- Secure invite system with 24-hour expiry codes
- Role-based access (Owner / Member)
- Consolidated reports across all family members
- Shared budget limits and alerts

### 5. Budget Limits & Alerts
Users set monthly spending caps per category. When approaching or exceeding a limit, Finstein sends a real-time warning inline with transaction confirmation — no separate alert needed.

### 6. Recurring Expense Automation
Fixed monthly bills (mortgage, car insurance, Netflix, etc.) can be registered once. Finstein automatically logs them on the correct day each month — zero manual entry.

### 7. Transaction History & Editing
- Full searchable history via `/history`
- Edit or delete any past transaction via inline buttons or natural conversation
  - "No, that was 85 not 58" → auto-detects edit intent
- Soft-delete with `/trash` — deleted transactions are recoverable
- `/undo` to instantly revert the last entry

### 8. Savings Goals & Wallet Tracking
- Multiple wallet/account balances (chequing, savings, investment)
- Savings goal creation with progress tracking
- Manual balance updates via conversation

### 9. Visual Analytics
- Pie charts by spending category
- Bar charts for monthly trends
- Generated on-demand via `/chart` command

### 10. Data Export
- Full transaction history exportable as `.xlsx` at any time
- Compatible with Excel, Google Sheets, any accounting software

## Technology Stack

### AI & Intelligence Layer
| Component | Model | Purpose |
|-----------|-------|---------|
| Transaction Parser | Claude Haiku 4.5 | Fast NL → structured data, <1s response |
| Financial Analyzer | Claude Sonnet 4.0 | Deep trend analysis, health scoring |
| Financial Advisor | Claude Sonnet 4.0 | Personalized monthly recommendations |
| Conversational QA | Claude Haiku 4.5 | Real-time spending queries |
| Voice Transcription | OpenAI Whisper | Speech → text for voice messages |

### Backend Architecture
- **Runtime**: Node.js 22 (TypeScript) — type-safe, production-grade
- **Bot Framework**: Grammy — modern, high-performance Telegram framework
- **Database**: PostgreSQL 16 with Prisma ORM — relational data integrity
- **Hosting**: Render.com — auto-scaling, managed PostgreSQL, zero-downtime deploys
- **Scheduler**: Node-cron for automated recurring expenses and monthly reports

### Data Models
- `User` — Telegram ID, language preference, premium status, timezone
- `Transaction` — amount, category, date, memo, soft-delete flag
- `FixedExpense` — recurring bills with day-of-month trigger
- `BudgetLimit` — per-category monthly caps
- `WalletAccount` — named account balances
- `Goal` — savings targets with progress
- `Family` — household group with invite codes

### Integrations
- **Telegram API** — native chat platform (900M+ users)
- **Anthropic API** — Claude models for parsing and analysis
- **OpenAI API** — Whisper for voice transcription
- **Google Charts API** — chart visualization
- **XLSX library** — Excel file import/export

## Security & Privacy

### Authentication
- Zero-password authentication — identity is verified by Telegram itself (industry-standard OAuth equivalent)
- Each user is uniquely identified by their immutable Telegram ID
- No passwords stored, no credential leaks possible

### Authorization & Data Isolation
- Every transaction is ownership-verified before edit/delete access
- Family data is fully isolated — no cross-family data access possible
- Role-based permissions: only Family Owners can invite/remove members
- Premium feature gating enforced server-side, not client-side

### Input Security
- All environment variables validated via Zod schema at startup
- File uploads restricted by type (.xlsx, .xls, .csv only) and size (5MB max)
- Category inputs whitelisted — no arbitrary data injection
- Rate limiting middleware prevents API abuse and spam

### Invite System Security
- Invite codes are 12-character hex strings (281 trillion combinations)
- 24-hour TTL — codes expire automatically
- Single-use — each code is consumed on first valid use
- Stored in-memory only — not persisted to database

### Data Integrity
- Soft-delete architecture — no permanent data loss without explicit user action
- Full transaction audit trail preserved
- Recoverable trash bin for all deleted records
